# Tours Search Frontend

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run typecheck`
- `npm run format`
- `npm run format:check`

- Vite

- TypeScript
- React Router
- TanStack Query

- ESLint
- Prettier
